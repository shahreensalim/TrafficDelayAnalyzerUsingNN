This project contains all the necessary codes for our project "Predicting Network Delays Using Knowledge-Defined
Networking Framework".
1. First go into the project folder.
2. requirements.txt contains all necessary libraries to run this project.
3. After installing everything from requirements.txt, to run the project, write "python3 app.py" on terminal and the project will start. You will be able to access the visual analytics tool by going into "http://127.0.0.1:5000/" from your browser.
3. Notebook Files folder contains all python notebooks (and their corresponding readme files). The python notebooks contain the implementation of the NN and generating result graphs.
